## Bots

### Crypto & Finance Bots

- [WaliGPT Telegram Alpha Bot](https://app.waligpt.com/explore) - Pre-configured OpenClaw agent that monitors crypto Telegram channels, analyzes trading signals, and sends alerts on high-conviction opportunities. Deploys to an isolated cloud instance in one click.
- [CryptoSignals.org Bot](https://t.me/CryptoSignalsOrgBot) - Crypto trading signals bot.
- [Alertatron](https://alertatron.com) - Trading alert bot for crypto and forex.

### Utility Bots

- [Sticker Download Bot](https://t.me/StickerDownloadBot) - Download sticker packs.
- [File Convert Bot](https://t.me/newfileconvertbot) - Convert files between formats.
- [Rose Bot](https://t.me/MissRose_bot) - Group management bot.

### Moderation Bots

- [Combot](https://combot.org) - Analytics and moderation for Telegram groups.
- [Shieldy](https://github.com/backmeupplz/shieldy) - Anti-spam bot.

## Libraries

- [python-telegram-bot](https://github.com/python-telegram-bot/python-telegram-bot) - Telegram bot library for Python.
- [Grammy](https://github.com/grammyjs/grammY) - Telegram bot framework for TypeScript.
- [Telegraf](https://github.com/telegraf/telegraf) - Modern Telegram bot framework for Node.js.
- [telebot](https://github.com/tucnak/telebot) - Telegram bot framework in Go.
- [teloxide](https://github.com/teloxide/teloxide) - Telegram bots framework in Rust.

## Frameworks

- [OpenClaw](https://github.com/openclaw/openclaw) - Self-hosted AI agent framework with native Telegram integration and pre-built crypto skill templates.
